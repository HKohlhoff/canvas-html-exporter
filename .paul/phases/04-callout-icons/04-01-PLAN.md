<objective>
## Goal
Add type-specific Unicode icons to callout title bars, matching Obsidian's visual style. Unknown/custom callout types get a generic fallback icon so they render correctly regardless of name.
</objective>

<context>
@canvas-exporter/src/converter.ts
</context>

<acceptance_criteria>

## AC-1: Standard callout types show type-appropriate icon
```gherkin
Given a callout of a known type (e.g. `> [!WARNING]`)
When exported to HTML
Then the title bar contains a `<span class="callout-icon">` with the appropriate Unicode character (e.g. ⚠)
```

## AC-2: Custom/unknown callout types show fallback icon
```gherkin
Given a callout with a non-standard type (e.g. `> [!settings]` or `> [!award]`)
When exported to HTML
Then the title bar contains `<span class="callout-icon">◆</span>` as fallback
```

## AC-3: Icon renders before title text
```gherkin
Given any callout
When exported
Then the icon span appears before the title text within `.callout-title` / `summary.callout-title`
```

</acceptance_criteria>
