<objective>
## Goal
Make callouts with `[!TYPE]+` (default open) and `[!TYPE]-` (default closed) collapsible in the exported HTML using native `<details>`/`<summary>` elements. Non-collapsible callouts (`[!TYPE]` without indicator) remain as `<div>` elements unchanged.

## Purpose
Obsidian supports collapsible callouts — users who use them expect the exported HTML to preserve that interactivity. The `+`/`-` indicator is already parsed but currently discarded.

## Output
Modified `canvas-exporter/src/converter.ts`:
- Regex captures the `+`/`-` indicator
- HTML output forks: `<div>` for non-collapsible, `<details>` for collapsible
- CSS in both style blocks handles `<details>`/`<summary>` reset and toggle indicator
</objective>

```gherkin

When exported to HTML

```

```gherkin

When the user clicks the title bar
Then the content expands or collapses
```

  <name>Task 1: Capture collapse indicator and fork HTML output</name>
  <files>canvas-exporter/src/converter.ts</files>
  <action>
    In `markdownToHtml()`, update the callout handler (around line 761–767):

    **Step 1 — Capture `[+-]` in regex:**
    ```
    // BEFORE:
    const calloutMatch = firstLine.match(/^\[!([\w-]+)\][+-]?(?:\s+(.*))?$/i);
    // AFTER:
    const calloutMatch = firstLine.match(/^\[!([\w-]+)\]([+-])?(?:\s+(.*))?$/i);
    ```
    Now: `calloutMatch[1]` = type, `calloutMatch[2]` = `+`/`-`/undefined, `calloutMatch[3]` = custom title.

    **Step 2 — Update title extraction** (index shifts from [2] to [3]):
    ```
    // BEFORE:
    const title = calloutMatch[2]?.trim() || (type.charAt(0).toUpperCase() + type.slice(1));
    // AFTER:
    const indicator = calloutMatch[2];
    const title = calloutMatch[3]?.trim() || (type.charAt(0).toUpperCase() + type.slice(1));
    ```

    **Step 3 — Fork HTML output based on indicator:**
    ```
    // BEFORE:
    out.push(`<div class="callout callout-${escapeAttribute(type)}"><div class="callout-title">${escapeHtml(title)}</div><div class="callout-content">${inner}</div></div>`);

    // AFTER:
    if (indicator === "+" || indicator === "-") {
      const openAttr = indicator === "+" ? " open" : "";
      out.push(`<details class="callout callout-${escapeAttribute(type)}"${openAttr}><summary class="callout-title">${escapeHtml(title)}</summary><div class="callout-content">${inner}</div></details>`);
    } else {
      out.push(`<div class="callout callout-${escapeAttribute(type)}"><div class="callout-title">${escapeHtml(title)}</div><div class="callout-content">${inner}</div></div>`);
    }
    ```

    Do not touch anything else in this function.
  </action>
