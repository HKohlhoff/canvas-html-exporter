Obsidian allows different ways to embed notes or only parts of them:
+ note as whole, referenced by filename
+ heading block, references by \#h1...6 
+ blocks, reverenced by blockids as \^id
and all of these can similarly used in canvas-files and will be equally transferred to the html-pages.
##### Remember: 
* **referencing** is just linking a note (or parts of it) in the master-note. The referenced content it not directly shown in the master-note. To read the content one needs to open the linked note via the link provided. 
* **embedding** is done the same way, but starts with an exclamation mark as first character of the link. But the main difference is, that the linked content is directly visible in the master-note as if it is written in there. 

To use references or contend-embedding this plugin uses the standard obsidian markdown notation. There is no "behind the scenes" logic or trick. So you need not to change anything in your existing notes or write new note in any other way than you did before.
